Forums

This page allows to create, modify, move or delete forums and sub-forums.

Create a new forum

You'll have to fill the following informations :

  • Global Access Level : choose the groups which will have access to this forum
  • Set forum position : set the order of the forums. 0 (zero) for the top forum
  • Give a name for your forum and a description
  • Category : choose the category for this forum in the dropdown list
  • Status : active - inactive ; you can here inactive a forum
  • Allow HTML : you can allow users to use HTML in their posts (very dangerous)
  • Allow signatures : while answering yes the signature recorded in the profile of the user will be included
  • Allow polls : if the xoopspoll module is installed and activated, the members authorized to create or answer a poll according to the authorizations which are granted to them.
  • Allow subject prefix for the topics : you can here allow the user to prefix the title of the thread with the parameters given in the module preferences
  • Hot topic threshold : level for considering a thread is hot (very important activity)
  • Allow attachments : allow users to send a file to their posts
  • Max Size in kb`s: Maximum size for the attachment in KB
  • Allowed extensions : you can choose the allowed extensions for the attachments
  • Topic permissions for this forum : for each action you can choose the allowed group

After creating the forum you'll have to select moderators

Others operations

  • modify a category or a forum
  • delete a category or a forum (and all contains)
  • move a forum or sub-forum in an other category
