Documentation for the Newbb module

 

This forum module allows creating categories where you can have forums and subsforums.

The new features in this new version are very important for administrators but also for users :

  • multilingual compatibility
  • printing as pdf file
  • polling capability
  • for each forum where you can :
    • print topic
    • move, delete, make sticky, lock, Make as digest a topic (depend of rights)
    • rate a thread
    • use a search form especialy for this module
    • modify the view mode (flat, thread, newest first)

  • in the main index you have an option button which permits :
    • Mark all forums read or unread
    • View all topics
    • View Digest topics
    • View unreplied topics
    • View unread topics (depend of the cookies)
