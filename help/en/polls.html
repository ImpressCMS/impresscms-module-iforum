Polls and votes

For having these functions you must have the xoopspoll module in your XOOPS (the standard module). Installed and ready to go.

You can see an indicator in the main admin index of the newbb2 module. It shows if polls are available.

In each forum you can choose witch group are allowed to use polls. (in a large way)

In the newbb2 admin area, the rating index shows polls and rating results.
