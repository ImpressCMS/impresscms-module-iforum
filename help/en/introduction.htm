Module installation

Fresh install

You don't have previous forum or you don't want to migrate old data ? follow this process :

  • download the module , but also the language file and the associated pictures
  • verify that your module folder don't have yet a newbb subfolder. (delete the folder newbb in this case)
  • decompress the archive contents in the modules folder of your site.
  • IMPERATIVE : replace your file /modules/system/admin/modulesadmin/main.php by this one : Correction for blocks update
  • modify your theme.html file and add just before the head tag : <{$xoops_module_header}>
  • install the module

Upgrade from a previous version

Newbb 1.1 -> Newbb 2.0 final

  • drop all the file in the old newbb folder
  • upload the files and run the appropriate script : newbb1_to_newbb2.php
  • update the module newbb in the "module admin" area
  • give access for categories and forums
  • for having functional menus, ensure the following line <{$xoops_module_header}> is present in your default theme.html.
  • update newbb's templates

Newbb 2.0 Rcx -> Newbb 2.0 final

  • upload the files and run the appropriated script
  • update the module newbb in the "module admin" area
  • update newbb's templates
