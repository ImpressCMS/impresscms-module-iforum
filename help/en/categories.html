Categories

Before creating forums, you should create at least one category

Create a new category

For this you'll have to fill the following options :

  • Global access level : choose the groups which will have access to this category
  • Set category position : set the order of the categories. 0 (zero) for the top category
  • Give a name for your category and a category description
  • Status : active - inactive ; you can here inactive a category
  • sponsor : you can give a partner link with an image

 

Other operations

In the list of categories you can of course Edit or Delete a category
