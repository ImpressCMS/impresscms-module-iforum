Documentation du module Newbb

 

Le module de forum Newbb2 permet de créer des catégories à l'intérieur desquelles il sera possible d'y insérer des forums et sous-forums.

Les nouvelles fonctionnalités que contient cette nouvelle version sont nombreuses tant pour les administrateurs du site que pour les utilisateurs :

  • compatible avec le hack multilanguages
  • impression d'un post au format pdf
  • possibilité de sondages
  • pour chaque sujet ou forum, il est possible de :
    • d'imprimer un sujet
    • déplacer, supprimer, verrouiller, agrafer, sommairiser un sujet (selon les droits accordés)
    • donner une note à un sujet
    • d'utiliser un formulaire de recherche spécifique à ce module
    • de modifier l'affichage par défaut (à plat, par conversation, emboîté)
  • sur la page index.php du module, un bouton options permet :
    • marquer tous les forums lus ou non lus
    • voir tous les sujets
    • voir les sujets sommairisés
    • voir les sujets sans réponse
    • voir les sujets non lus (basés sur les cookies du navigateur utilisé)
