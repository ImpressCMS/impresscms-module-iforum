Sondages et votes

Pour que cette fonction soit opérationnelle il faut avoir préalablement installé et activé le module standard xoopspoll livré avec la version standard de Xoops.

Un indicateur sur la page index.php dans l'interface d'administration du module newbb vous confirmera que cette option est possible.

En éditant chaque forum vous pourrez déterminer quels sont les groupes autorisés à utiliser (au sens large) les sondages.

Dans l'interface d'administration du module, l'onglet votes vous permet de visualiser les membres ayant participé à des sondages. (??)
