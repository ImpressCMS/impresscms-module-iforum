Les blocs

En standard ce module est livré avec les blocs suivants :

  • sujets récents
  • sujets les plus vus
  • sujets les plus actifs

A partir de cette page il vous est possible de les éditer pour modifier les attributs de ceux-ci :

  • emplacement du bloc : choix dans la liste déroulante
  • visible : oui - non
  • affichage sur quelle(s) page(s)
  • modification éventuelle du titre
  • paramétrage de différentes options :
    • nombre de titres de messages à afficher dans le bloc
    • mode de visualisation (complet, compact, léger)
    • possibilité de choisir des messages dans tous les forums ou seulement sur certains
    • temps de cache

Autorisations

La seconde partie de cette page est consacrée aux autorisations sur ces blocs. En une seule fois, vous aurez la capacité d'autoriser aux différents groupes :

  • les droits d'administration du module
  • les droits d'accès au modules
  • l'accès aux différents blocs

Création de blocs à la volée

Il est possible de créer des blocs personnalisés sur le forum :

  • Menu Administration'=>"Administration des blocs"=>"Ajouter un bloc
  • type de bloc : format php
  • poids : entrer une valeur de votre choix
  • visible : choisissez ce que vous désirez
  • visible dans : définissez les endroits où sera visible ce bloc
  • titre : donnez un titre à votre bloc
  • Contenu :
    • include_once(XOOPS_ROOT_PATH . '/modules/newbb/blocks/newbb_block.php');
    • $options = "10|0|time|0";
    • b_newbb_custom($options);
