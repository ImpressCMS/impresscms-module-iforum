Installation du module

Nouvelle installation

Vous démarrez de zéro et n'avez pas de migration à faire, voici la procédure à suivre :

  • téléchargez le module , sans oubliez de récupérer également le fichier langage français avec les images associées
  • vérifiez que votre répertoire modules ne contient pas déjà un dossier newbb, si c'est le cas supprimez-le
  • décompressez le contenu du fichier dans le répertoire modules de votre site
  • IMPERATIF : remplacez votre fichier /modules/system/admin/modulesadmin/main.php par celui ci : Correction pour la mise à jour des blocs
  • modifier votre fichier theme.html et intégrer ceci avant le head : <{$xoops_module_header}>
  • installer le module

Mise à jour d'une version antérieure

Newbb 1.1 -> Newbb 2.0 finale

  • supprimez tous les fichiers du répertoire de l'ancien newbb
  • upload des nouveaux fichiers et exécution du script approprié : newbb1_to_newbb2.php
  • mise à jour du module NewBB dans l'administration des modules
  • Définir les permissions pour les catégories et les forums
  • Pour que les menus fonctionnent, assurez-vous que cette mention <{$xoops_module_header}> est incluse dans votre fichier theme.html de votre thème par défaut.
  • mettre à jour les templates de newbb

Newbb 2.0 Rcx -> Newbb 2.0 finale

  • upload des nouveaux fichiers et exécution du script approprié
  • mise à jour du module NewBB dans l'administration des modules
  • mettre à jour les templates de newbb
